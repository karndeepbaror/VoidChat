
# 🔒 Security & Privacy — VoidChat

> *How anonymity works — and what it doesn't do.*

---

## ✅ What VoidChat DOES

| Feature | Status |
|---------|--------|
| No login / signup | ✅ |
| No message storage | ✅ |
| No IP logging | ✅ |
| No cookies / trackers | ✅ |
| Random anonymous identity | ✅ |

---

## ❌ What VoidChat Does NOT Do

| Feature | Reality |
|---------|---------|
| End‑to‑end encryption | ❌ Not yet |
| Hide you from ISP | ❌ Use Tor / VPN |
| Anonymize your network | ❌ IP passes through |
| Delete messages remotely | ❌ Only local |

---

## 🛡️ Recommendations for Maximum Privacy

- ✅ Use **Tor Browser** + **VoidChat**
- ✅ Use a **VPN** before connecting
- ✅ Never share personal info in chat
- ✅ Close browser after session

---

> 🔥 **VoidChat is a tool — use it wisely.**
